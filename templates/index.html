Descripción del Problema

La capacidad instalada en los prestadores de servicios de salud es un elemento fundamental para comprender cómo se distribuyen los recursos dentro de la red hospitalaria. Analizar esta información permite identificar diferencias entre prestadores según el tipo de servicio habilitado y la cantidad disponible.

Debido al alto volumen de registros, identificar patrones de manera manual resulta complejo. Por esta razón, es útil aplicar técnicas de agrupamiento que permitan clasificar automáticamente los prestadores de acuerdo con características similares.

Por ende para este trabajo se utilizan el algoritmo K-Means Clustering con el propósito de agrupar prestadores de servicios de salud a partir de variables relacionadas con la capacidad instalada, permitiendo detectar comportamientos similares y patrones dentro del conjunto de datos.

Objetivo

Identificar agrupamientos de prestadores de servicios de salud con características similares en términos de capacidad instalada, utilizando el algoritmo K-Means para detectar patrones dentro del conjunto de datos y generar una base de apoyo para el análisis, la planeación y la toma de decisiones relacionadas con la distribución de recursos y la prestación de servicios.
