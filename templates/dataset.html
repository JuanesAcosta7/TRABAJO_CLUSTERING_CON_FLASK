Descripción del Dataset

Para el desarrollo del ejercicio se utilizó un dataset relacionado con la capacidad instalada en la red de prestación de servicios de salud. El archivo contiene información sobre prestadores, grupos de capacidad habilitada, códigos de capacidad y cantidades registradas dentro del sistema.

El dataset empleado contiene aproximadamente 5.461 registros, lo cual permite aplicar técnicas de agrupamiento sobre un volumen suficiente de datos para identificar patrones de comportamiento entre prestadores de servicios de salud.

Variables Utilizadas

Nombre del Prestador: identifica el prestador de servicios de salud registrado.

Grupo Capacidad: clasifica el tipo general de capacidad habilitada.

Código Capacidad: representa de forma numérica el tipo específico de capacidad registrada.

Cantidad: indica el volumen de capacidad instalada reportado por el prestador.

Preparación de Datos

Como parte del proceso de preparación, se seleccionaron únicamente las variables relevantes para el análisis. Posteriormente, se eliminaron registros con valores nulos y se transformó la variable categórica correspondiente al grupo de capacidad en valores numéricos, con el fin de permitir su utilización dentro del algoritmo K-Means.

Fuente del Dataset

El dataset utilizado corresponde a información pública relacionada con capacidad instalada de la red de prestación de servicios de salud.
