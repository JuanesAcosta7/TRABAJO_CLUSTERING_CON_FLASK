Modelo K-Means

En este proyecto se utilizó el algoritmo K-Means Clustering, una técnica de aprendizaje no supervisado que permite agrupar registros con características similares.

El algoritmo funciona asignando cada registro al centroide más cercano, formando grupos de datos internamente similares y externamente diferentes respecto a otros clusters.

Conceptos Principales

Cluster: conjunto de registros con características similares.

Centroide: punto central que representa cada agrupamiento.

Distancia: criterio utilizado para asignar cada registro al cluster más cercano.

Ejecución del Modelo

Para la ejecución del modelo se seleccionaron variables relacionadas con la capacidad instalada de los prestadores de servicios de salud: código de capacidad, cantidad y grupo de capacidad.

Posteriormente, los datos fueron depurados eliminando registros nulos y transformando variables categóricas en valores numéricos.

Finalmente, se aplicó un proceso de normalización mediante StandardScaler y se ejecutó el algoritmo K-Means con un valor inicial de K igual a 3.

Método del Codo

Para justificar el número de clusters utilizado, se aplicó el método del codo, el cual permite evaluar la inercia del modelo para diferentes valores de K y detectar el punto donde la disminución deja de ser significativa.

Aplicación del Método del Codo

Se calculó la inercia del modelo para distintos valores de K con el fin de analizar la variación interna de los agrupamientos.

La siguiente gráfica muestra el comportamiento de la inercia y permite identificar visualmente el valor más adecuado para el número de clusters.

A partir de la gráfica del método del codo se observa que la disminución de la inercia es pronunciada entre los primeros valores de K, pero comienza a estabilizarse a partir de K = 4.

Esto indica que utilizar cuatro clusters permite captar una buena parte de la estructura interna de los datos sin generar una segmentación innecesariamente compleja. Por esta razón, se seleccionó K = 4 como el número más adecuado de agrupamientos para el modelo.
